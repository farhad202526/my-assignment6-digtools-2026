

1. Name of your Project
DigiTools Premium – Your Ultimate Digital Solutions Hub

2. A little Description
DigiTools Premium is a modern, responsive web application designed for career seekers and professionals. It provides a curated collection of digital products and pricing plans (Starter, Professional, Enterprise, etc.) to help users boost their productivity. Users can browse various service cards, see detailed features, add them to their cart, and proceed to a seamless checkout experience.

3. Technology that you use 

React.js: For building the dynamic user interface.

Tailwind CSS: For modern and responsive styling.

DaisyUI: For pre-designed, beautiful UI components (Buttons, Cards, etc.).

React-Toastify: To show elegant notifications (Toasts).

React Icons: For high-quality vector icons.

JSON (Local API): To manage and fetch product data dynamically.

Vite: As the fast development server and build tool.

4. 3 Features about your project
Dynamic Pricing & Category Management:
The app fetches data from a JSON file and displays 6 different categories of plans. Each plan has a dynamic "Tag" (e.g., Popular, New, Lifetime) with unique background colors (Blue, Red, Green, etc.) based on the category type.

Interactive Shopping Cart with Validation:
Users can add products to their cart with a single click. The "Buy Now" button automatically changes to "Added to Cart" and becomes disabled to prevent duplicate entries, ensuring a smooth user experience.

Real-time Notifications & Checkout Logic:
Integrated with React-Toastify, the app provides instant feedback when a user interacts with the cart. The "Proceed to Checkout" feature allows users to clear their entire cart at once, resetting all product buttons back to their original state.
